# My Discord Theme

Rust Viridian is a customized Discord theme built on top of the **Midnight** theme by [refact0r](https://www.refact0r.dev).

---

## Description

- **Color scheme for my taste**  
  Also included several CSS snippets from Discord.

- **Various snippets for better UX**
  Including Spotify player etc.

- **Horizontal Server List**  
  Aligned the HorizontalServerList for the midnight theme layout.

---

## Installation (Vencord)

1. Place `rustviridian.css` into your Vencord **themes** folder.
2. Place the addon file `./addons/HorizontalServerList.theme` into the same themes directory.
